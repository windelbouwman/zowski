
# About

Scanner generator using regular derivatives (Brzozowski derivatives).

Usage example:

    $ cargo run
    $ dot -Tps machine.dot -o machine.ps
    $ okular machine.ps

Status: idea

# Features

- unicode support
- regex `not` operator
